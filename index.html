INTRODUCCION A HTML

Significa hipertexto, es un programa que define la estructura de una pagina web, los elemento de HTML le indican al navegador como mostrar el contenido.

Elementos Html

Encabezados:
Sirven para agregar un titulo

Ejemplos de encabezados
Parrafos:
Estos permiten agregar un texto

Ejemplos de parrafos
Listas:
Agrupan una cantidad de elementos en una lista, estas pueden estar organizadas o desorganizadas

Listas organizadas:

  1. elemento a
  2. elemento b
  3. elemento c
  4. elemento d

Listas desorganizadas:

  • elemento d
  • elemento i
  • elemento v
  • elemento x

Ejemplos de listas
Enlaces:
Permiten agregar el url de un objeto

Ejemplos de enlaces
Imagenes:
Permiten mostrar una imagen
